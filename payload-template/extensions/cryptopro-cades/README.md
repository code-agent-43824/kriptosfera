# CryptoPro CAdES extension scaffold

Этот каталог зарезервирован под unpacked extension payload для этапа `v0.4`.

Пока здесь только scaffold:
- layout для доставки extension внутри payload;
- стабильная точка интеграции для launcher-side detection;
- место для будущих реальных extension files.

Следующим шагом сюда будет добавлен канонический набор файлов extension и проверка его загрузки в Chromium runtime.
